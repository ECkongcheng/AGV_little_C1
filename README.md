# AGV_little_C1
基于ROS melodic，底盘控制器基于STM32的一个2D激光SLAM自主导小车 https://www.bilibili.com/video/BV12J411e7uH
## 说明
工控机：Intel_up_core,底盘控制器：STM32F103ZETA6，STM32F103C8T6,传感器：USB2.0_720PCamera 思岚A1 激光雷达
